# Website Institucional - Tavares Tech Hub

Este projeto representa a vitrine digital da **Tavares Tech Hub**, focado em oferecer soluções tecnológicas com agilidade e eficiência. 

## 🚀 Tecnologias Utilizadas
- **HTML (HyperText Markup Language)**: Estruturação de conteúdo.
- **CSS (Cascading Style Sheets)**: Identidade visual e layout mobile-first.
- **JavaScript**: Interatividade e comportamento dinâmico.

## 🛠️ Metodologia
Desenvolvido através de **Metodologias Ágeis** e com o suporte de **Inteligência Artificial Generativa** para otimização de código e design, refletindo meu estágio atual de atuação técnica.

# 🚀 Tavares Tech Hub — Tabela de Serviços

> Site profissional de apresentação de serviços com comparativo de preços em relação ao mercado.

---

## 📌 Sobre o Projeto

O **Tavares Tech Hub** é um projeto de empreendedorismo individual que reúne mais de **40 serviços** nas áreas de consultoria empresarial, assistência técnica em software, defesa do consumidor e serviços de papelaria.

Este repositório contém o site de apresentação do portfólio de serviços, desenvolvido em **HTML e CSS puro**, sem frameworks externos, com foco em clareza, profissionalismo e transparência de preços.

---

## ✨ Diferenciais do Projeto

- 💰 **Comparativo de preços em tempo real** — cada serviço exibe o valor cobrado vs. o praticado pelo mercado, mostrando ao cliente o real benefício financeiro
- 📱 **100% responsivo** — funciona perfeitamente em celulares, tablets e computadores
- ⚡ **Leve e rápido** — HTML e CSS puro, sem dependências externas
- 🎨 **Design profissional** — paleta de cores coesa, tipografia limpa e tabelas bem organizadas
